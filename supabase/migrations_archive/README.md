# Archived Migrations

これらのマイグレーションファイルは、既に適用済みで今後の新規DB作成時には不要なファイルです。

## アーカイブ理由

- **テストユーザー関連**: 開発中のテストユーザー作成/削除の試行錯誤
- **パスワードリセット関連**: 認証問題の修正試行
- **重複ファイル**: 既存マイグレーションの重複コピー
- **デモデータ関連**: 一時的なデモデータ投入スクリプト
- **修正試行**: 同じ問題に対する複数の修正試行

## 現在のDBへの影響

なし。これらのファイルは既に適用済みで、アーカイブしても現在のDBには影響しません。

## 新規DB作成時

アーカイブされたファイルは無視され、コアマイグレーションのみが適用されます。
